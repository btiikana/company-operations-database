# Island Supply Company Operations Database System

This repository documents a project-based database engineering learning path focused on building a complete company operations database system.

The project is built around the Island Supply Company Operations Database System, a simulated business database used to practice business analysis, requirements thinking, data modeling, relational database design, SQL, PostgreSQL, technical documentation, database administration, security, backup and recovery, and Python automation.

## Project Goal

The goal of this project is to learn how to move from business needs to a working relational database system.

This project follows one core rule:

Every skill must build a piece.
Every piece must create an artifact.
Every artifact must become part of the final Island Supply Company system.

## Course Platform

The course platform used for this learning path is Coursera.

Courses are not completed blindly from start to finish. Each course is filtered based on the current project piece and the artifact that needs to be built.

## Repository Structure

* `docs/` - project documentation, requirements, business rules, design notes, and guides
* `diagrams/` - process diagrams, ERDs, and visual models
* `sql/` - SQL scripts for schema creation, seed data, queries, constraints, reports, and tests
* `data/` - sample CSV data
* `python/` - Python scripts for loading data, reports, alerts, and automation
* `progress/` - learning path, course tracking, piece tracking, decisions, and daily logs
* `certificates/` - course certificates and related proof of learning
* `backups/` - backup and restore examples
